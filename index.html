Pearl Loop

A quiet logic puzzle about black pearls, white pearls, and one elegant continuous path.

Draw the Perfect Loop

Pearl Loop is an offline puzzle game inspired by classic loop logic. Each board asks you to draw a single closed path through the grid while respecting the special rules of black and white pearls. It starts gently, then grows into a satisfying test of planning, pattern recognition, and clean deduction.
